\begin{soal}[Chapra Latihan 8.33]
Impedansi dari rangkaian paralel RLC dinyatakan oleh persamaan
\begin{equation*}
\frac{1}{Z} = \sqrt{\frac{1}{R^2} + \left(
\omega C - \frac{1}{\omega L}
\right)^2}
\end{equation*}
Cari frekuensi angular $\omega$ untuk $Z=75$ ohm,
$R=225$ ohm,
$C = 0.6 \times 10^{-6}$ F
dan $L = 0.5 H$.
\end{soal}
